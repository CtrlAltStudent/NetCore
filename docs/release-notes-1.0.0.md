# NetCore 1.0.0 – Release notes (planowane)

Pierwsza stabilna wersja NetCore. Planowane po zebraniu uwag z wersji beta.

## Zakres (planowany)

- Wszystkie funkcje z 0.9.0-beta.1.
- Poprawki po testach beta (bezpieczeństwo, wydajność, UX).
- Ewentualne rozszerzenia: wybór okresu na dashboardzie, wykresy (trendy), rejestracja z poziomu aplikacji MAUI.

## Wersjonowanie

- **Beta:** 0.9.0-beta.1, 0.9.0-beta.2, …
- **Stabilna:** 1.0.0 (SemVer: MAJOR.MINOR.PATCH).
