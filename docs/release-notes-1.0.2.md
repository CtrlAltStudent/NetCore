# NetCore 1.0.2 – Release notes

**Data wydania:** 2026-02-06

## Zakres

- **API:** Endpoint szeregu czasowego raportów (`/api/v1/reports/series`) do wykresów.
- **MAUI:** Nowe zakładki – Pracownicy (CRUD), Reguły premii (CRUD).
- **MAUI:** Wykresy (LiveCharts2): trend na Dashboardzie i w Raportach, wykres kołowy przychodów per kanał; na Windows – komunikaty zamiast wykresów.
- **MAUI:** Wybór okresu na Dashboardzie, karty KPI, wskaźniki ładowania.
- **Poprawki Windows:** Czarny/gradientowy ekran (Mica wyłączone, białe tło okna); błędy bootstrapu na tej samej stronie; `DisplayAlertAsync` zamiast `DisplayAlert`.
- **Dokumentacja:** Rozszerzona instrukcja uruchomienia (Cursor, build-maui, typowe problemy).

## Wersjonowanie

- **Wersja:** 1.0.2 (SemVer: PATCH – nowe funkcje i poprawki).
